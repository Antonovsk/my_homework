# Подсказки по командной строке

Команда смены директории 
```sh
cd c:\folder_name
```

Команда отображения текущей директоии. MacOs, Linux

```sh
pwd 
```

Листинг текущей директории
Windiws:
```sh
dir
```
Linux, MacOs:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
в Linux, MacOs:
```sh
rm <filename>
```
